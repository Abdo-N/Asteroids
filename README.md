# 🪐 Asteroids

A simple asteroid game made with `pygame`.

> Built as part of the [boot.dev](https://boot.dev) curriculum — dodging space rocks, one frame at a time. 🚀

---

## 🎮 About

This is a classic-style Asteroids clone where you pilot a small ship through a field of tumbling asteroids, blasting them apart before they get you first.

## 🛠️ Built With

- [Python](https://www.python.org/)
- [Pygame](https://www.pygame.org/)
